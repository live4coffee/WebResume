 AWS Mythical Mysfits

    AWS Tutorial - Build a Modern Web Application

  • 1. Build a static website, using Amazon Simple Storage Service (S3) that serves static content (images, static text, etc.) for your website.
  • 2. Host your application logic on a web server, using an API backend microservice deployed as a container through AWS Fargate.
  • 3. Externalize all of the mysfit data and persist it with a managed NoSQL database provided by Amazon DynamoDB.
  • 4. Enable users to registration, authentication, and authorization so that Mythical Mysfits visitors can like and adopt myfits, enabled through AWS API Gateway and its integration with Amazon Cognito.
  • 5. Capture user behavior with a clickstream analysis microservice that will record and analyze clicks on the website using AWS Lambda and Amazon Kinesis Firehose.
